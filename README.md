## PDF出力
- 1. ``ptex2pdf``をインストール

- 2. 以下のコマンドを実行
```sh
$ make export
```

## 素材書き出し
- 1. npmパッケージをインストール
```sh
$ npm install
```

- 2. Sketch**3.5**以下の時のみ``sketchtool``をインストール
```sh
$ sudo sh ./install-sketchtool.sh
```


- 3. 以下のコマンドを実行
```sh
$ npm run export
```
