## PDF出力
- １. ``ptex2pdf``をインストール

- ２. 以下のコマンドを実行
```sh
$ make export
```

## 素材書き出し
- １. npmパッケージをインストール
```sh
$ npm install
```

- ２. Sketch**3.5**以下の時のみ``sketchtool``をインストール
```sh
$ sudo sh ./install-sketchtool.sh
```

- ３. 以下のコマンドを実行
```sh
$ npm run export
```
